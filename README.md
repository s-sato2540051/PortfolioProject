ポートフォリオ共有サイト 概要

目的
クリエイターや開発者が自身の作品（ポートフォリオ）を投稿・共有できるサイト。
他のユーザーの作品を閲覧・評価したり、共同制作者をタグ付けしてチーム制作の実績をまとめることができる。

【機能一覧】

〇アカウント関連
・新規登録 / ログイン / ログアウト機能
・プロフィール編集（アイコン、自己紹介、外部リンクなど）
・各ユーザーのポートフォリオ一覧ページ表示

〇 ポートフォリオ投稿機能
・新規投稿
　- タイトル
　- 概要（作品説明）
　- サムネイル画像（1枚）
　- 追加画像（最大4枚）
　- タグ・ジャンル機能
　　例：#映像作品、#3Dモデリング、#Web制作 (DjangoのManyToManyFieldを使用)
　- 共同制作者の登録
　　サイト内のアカウント、またはYouTube/Xアカウントを入力可能
　　名前を入力すると類似アカウントをリアルタイムで表示（Ajaxを使用）
　- 外部リンク（作品ページや動画URLなど）

・編集 / 再投稿
　- 投稿後の内容や画像を編集可能
　- 公開 / 非公開 の切り替え機能

〇 閲覧機能
・全作品一覧ページ（タグ、ジャンル、キーワード検索付き）
・個別ポートフォリオページ（サムネイル、説明、共同制作者、タグ表示）
・ユーザープロフィールページ（その人の全投稿を一覧表示）
・タグをクリックすると同タグの作品一覧を表示

〇 いいね機能
・各作品に「いいね」ボタンを設置
・自分が「いいね」した作品を一覧表示できるページ
